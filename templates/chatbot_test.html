🤖 Nyveralife Chatbot Test

✅ Test Instructions

  1. Look for the green chat button in the bottom-right corner
  2. Click it to open the chatbot window
  3. Try these test messages:
    • • "আপনার কাছে কলা আছে?" (Bangla test)
    • • "Do you have bananas?" (English test)
    • • "কলা অর্ডার করতে চাই" (Buying intent test)
    • • "Delivery কত দিনে হয়?" (Support test)
  4. Check if you get intelligent responses

📊 Expected Behavior:

  • ✅ Chatbot should respond in the same language you use
  • ✅ Should show product info with prices and stock
  • ✅ Should detect if you want to buy (enthusiastic response)
  • ✅ Should provide delivery and payment info
  • ✅ Responses should be smart and sales-focused

🐛 If Chatbot Doesn't Work:

  1. Press F12 to open Developer Console
  2. Click on "Console" tab
  3. Look for any red error messages
  4. Try sending a message and check for errors
  5. Take a screenshot and share with me
